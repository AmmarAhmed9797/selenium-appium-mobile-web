---

## 📌 About This Project

Built at **Digital Landscape Pvt. Ltd.** delivering automated testing across 6 diverse projects. This framework covers both web automation with Selenium WebDriver and mobile automation with Appium, along with performance testing via JMeter.

### Key Achievements
- ✅ Automated testing across 6 projects: CRM, Healthcare, LMS, ERP-eCommerce
- ✅ Web + Mobile coverage using Selenium WebDriver & Appium
- ✅ Performance testing under peak load using JMeter
- ✅ Key contributor to Novartis Oncology Access Program (NOAP) QA delivery
